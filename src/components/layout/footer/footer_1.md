# Footer 1

## Variables

The following variables are needed in a JSON file named `footer.json` in the `defaults` folder:

```json
{
  "company": {
    "name": "Meine Firma GmbH",
    "url": "/impressum"
  },
  "navigation": [
    { "title": "Seite 11", "url": "/seite-1" },
    { "title": "Seite 22", "url": "/seite-2" },
    { "title": "Seite 33", "url": "/seite-3" }
  ]
}
```
