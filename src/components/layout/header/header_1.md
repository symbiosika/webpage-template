## Variables

The following variables are needed in a JSON file named `header.json` in the `defaults` folder:

```json
{
  "logoPath": "/images/logo.png",
  "logoAlt": "Firmenlogo",
  "logoText": "Logo",
  "logoUrl": "/",
  "menuItems": [
    {
      "text": "Seite 1",
      "url": "/seite-1"
    },
    {
      "text": "Seite 2",
      "url": "/seite-2"
    },
    {
      "text": "Kontakt",
      "url": "/seite-1"
    }
  ]
}
```
